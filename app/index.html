Planejamento Roteiro API

Código disponível: Github

API Disponível: API

Método POST para planejar roteiros no endereço: http://webroteiro.appspot.com/api/custo

Técnologias utilizadas: Flask,Github,Google App Engine

Linguagem utilizadas: Python

Exemplos de execução com cURL

curl -H "Content-Type: application/json " -X POST -d '{"tarefas":[ {"identificador":"a","tempo_de_execucao":420,"custo_por_hora":200},{"identificador":"b","tempo_de_execucao":120,"custo_por_hora":100},{"identificador":"c","tempo_de_execucao":1200,"custo_por_hora":2},{"identificador":"d","tempo_de_execucao":1800,"custo_por_hora":4}, {"identificador":"e","tempo_de_execucao":300,"custo_por_hora":8}]}' http://webroteiro.appspot.com/api/custo

Saída esperada: {"menor_tempo_espera": ["b", "e", "a", "c", "d"], "menor_custo_espera": ["b", "a", "e", "d", "c"]
